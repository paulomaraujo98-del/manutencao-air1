## Estrutura

- `index.html` — dashboard
- `test.html` — página de testes manuais
- `css/` — estilos
- `js/` — código do frontend
- `apps-script/` — código do backend (Google Apps Script)
- `docs/` — guias de setup e validação

## Para rodar localmente

Abra `index.html` num navegador (Chrome/Edge). Não precisa de servidor.
